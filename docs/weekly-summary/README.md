# Weekly Summary

Interactive dashboard summarizing kgents development work.

## View

Open `index.html` in any modern browser:

```bash
open docs/weekly-summary/index.html
```

Or serve locally:

```bash
python3 -m http.server 8000 --directory docs/weekly-summary
# Then visit http://localhost:8000
```

## Features

- **Work Streams**: Click cards to expand details (phases, features)
- **Timeline**: Chronological view of major milestones
- **Dependencies**: Visual dependency graph between plans
- **Commits**: Searchable commit history
- **Statistics**: Test counts, progress metrics
