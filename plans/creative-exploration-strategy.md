## Session Template

Each session follows this rhythm:

```
1. GROUND (5 min)
   - What did we celebrate last session?
   - What's the energy level today?

2. READ (15 min)
   - Deep dive into the spec(s)
   - Note what's implemented vs theoretical
   - Find the "wow" moments in the philosophy

3. BRAINSTORM (30 min)
   - Generate 10+ ideas per agent
   - No filtering yet
   - Cross-reference with other agents
   - Web research for validation/inspiration

4. EVALUATE (10 min)
   - Score each idea on 4 dimensions
   - Identify quick wins vs bigger builds

5. CRYSTALLIZE (10 min)
   - Top 3 ideas get detailed writeups
   - Add to project evaluation matrix
   - Note dependencies and blockers

6. CLOSE (5 min)
   - What energized us?
   - What's the one thing to marinate on?
```

## Project Evaluation Matrix

### The Four Dimensions

| Dimension | Question | Scale |
|-----------|----------|-------|
| **FUN** | Would this make someone smile? | 1-5 (1=meh, 5=delightful) |
| **EFFORT** | How hard to build? | 1-5 (1=hours, 5=weeks) |
| **SHOWABLE** | Can you demo this in 30 seconds? | 1-5 (1=needs context, 5=instant wow) |
| **PRACTICAL** | Does this solve a real problem? | 1-5 (1=pure art, 5=daily use) |

### Scoring Guide

**FUN**:
- 5: Laugh out loud, want to share immediately
- 4: Smile, want to explore more
- 3: Interesting, would use
- 2: Useful but dry
- 1: Necessary but boring

**EFFORT**:
- 1: < 2 hours (one sitting)
- 2: Half a day
- 3: 1-3 days
- 4: 1-2 weeks
- 5: Weeks+, needs architecture

**SHOWABLE**:
- 5: Screenshot/GIF tells the whole story
- 4: 30-second demo sufficient
- 3: Needs 2 minute explanation
- 2: Needs context and setup
- 1: Only impressive to experts

**PRACTICAL**:
- 5: Would use daily
- 4: Would use weekly
- 3: Occasional use case
- 2: Niche but valuable
- 1: Art/philosophy piece

### Priority Formula

```
PRIORITY = (FUN √ó 2 + SHOWABLE √ó 2 + PRACTICAL) / (EFFORT √ó 1.5)
```

Higher priority = better ROI on joy.
