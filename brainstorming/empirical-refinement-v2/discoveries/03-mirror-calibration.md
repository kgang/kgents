## Design Notes

### Why These 20 Items?

The items were selected for diversity across:

1. **Formality spectrum**: From pure math (Item 1, 6) to pure feeling (Item 3, 20)
2. **Content type**: Definitions, claims, principles, metaphors, design rules
3. **Ambiguity levels**: Some clearly categorical, some clearly somatic, many in between
4. **Source variety**: CLAUDE.md, Constitution, specs, theory docs

### Loss Estimation Methodology

Estimated loss was computed using:

```
loss = 0.5 * (1 - structural_clarity) +
       0.3 * (1 - formality) +
       0.2 * (1 - reproducibility)

where each factor is rated 0-1:
- structural_clarity: Can independent reviewers decompose it the same way?
- formality: Is it stated in mathematical/logical form?
- reproducibility: Would experts agree on the same conclusion?
```

This is a **heuristic**, not the true Galois loss. The true loss requires LLM restructuring. But the heuristic should correlate if the theory is sound.
