## The Three Pillars

### Pillar 1: Loss-Native Architecture

Every UI element carries a Galois loss signature. This is not decorative—it is the fundamental quality signal.

```
Low Loss (L < 0.2)  → Purple glow, stable, axiomatic
Mid Loss (L = 0.5)  → Teal, transitional, evolving
High Loss (L > 0.7) → Yellow pulse, unstable, investigate
```

**Implication**: Technical debt becomes visible. Orphaned components glow. Weak bindings pulse.

### Pillar 2: Telescope-as-Shell

There are no "pages"—only focal distances on the hypergraph:

```
∞       → Axiom constellations (L1)
100     → Specification trees (L4)
0       → Full editor (L7)
```

Navigation is loss-gradient descent: `gl` moves toward stability, `gh` investigates instability.

### Pillar 3: Four-Colored Analysis

Every node can be analyzed through four lenses simultaneously:

```
┌─────────────────┬─────────────────┐
│  CATEGORICAL    │  EPISTEMIC      │
│  Laws hold? ✓   │  Grounded? ✓    │
├─────────────────┼─────────────────┤
│  DIALECTICAL    │  GENERATIVE     │
│  Tensions: 2    │  Regenerable? ✓ │
└─────────────────┴─────────────────┘
```

This replaces the binary "valid/invalid" with rich understanding.

## The Migration Strategy

### Phase 1: Loss Awareness (Week 1)
Add `useLoss()` hook, `<LossBadge />` component, viridis palette.
**Measurable**: Every ZeroSeed node shows its loss.

### Phase 2: Telescope Shell (Week 2-3)
Extract `useTelescopeState()`, wrap in `<TelescopeShell />`, add focal distance controls.
**Measurable**: Users can zoom L1↔L7 without page switches.

### Phase 3: Gradient Navigation (Week 3)
Add `gl`/`gh` commands, gradient field overlay, loss threshold slider.
**Measurable**: Users navigate toward stability, not arbitrary browsing.

### Phase 4: Analysis Quadrants (Week 4)
Create four-panel analysis view, invoke via `<leader>a`.
**Measurable**: Every node shows all four analysis modes.

### Phase 5: Witness Integration (Week 5)
Refactor marks as edges, add `gm`/`gw`/`gd` navigation.
**Measurable**: Decisions are navigable, not just logged.
